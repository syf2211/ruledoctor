# 本地使用（简版）

完整说明请读 **[用户指南.md](用户指南.md)**。

## 三步

1. **装 Skill** → `cp -R skills/ruledoctor ~/.claude/skills/`（Codex/Cursor 同理）
2. **新开对话**，在有 `CLAUDE.md` 或 `.cursorrules` 的项目里试一个小任务
3. 看助手是否**先提规则**；可选 `ruledoctor --last-session` 自查（见 [分数是什么意思.md](分数是什么意思.md)）

## 可选 CLI

```bash
git clone https://github.com/syf2211/ruledoctor.git
cd ruledoctor && npm install && npm run build
node dist/index.js setup -p your-project
```
