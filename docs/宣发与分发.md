## 2. 官方 / 半官方收录路径

### 2.1 skills.sh（Vercel 生态）

| 项目 | 内容 |
|------|------|
| **是什么** | 开放 Skill 目录 + `npx skills` CLI；CC Switch、60+ agent 共用安装器 |
| **收录方式** | 公开 GitHub 仓 + 符合 `skills/<name>/SKILL.md`（或仓内可被 CLI 发现的 skill）；**自动出现在 skills.sh** |
| **我们缺什么** | 安装量、README 上的 badge、可选 `homepage` in package.json |
| **动作** | README 加 `[![skills.sh](https://skills.sh/b/syf2211/ruledoctor)](https://skills.sh/syf2211/ruledoctor)`；鼓励用户用官方 install 命令（计入 leaderboard） |

### 2.2 Claude Code Plugin Marketplace

| 项目 | 内容 |
|------|------|
| **是什么** | 仓内 `.claude-plugin/marketplace.json` + `plugin/`（**含 Hooks**，比纯 Skill 重） |
| **收录方式** | 用户本地：`/plugin marketplace add syf2211/ruledoctor` → `/plugin install ruledoctor@syf2211-ruledoctor` |
| **官方精选** | `anthropics/skills` 等为官方策展；第三方 **无公开「提交审核」表单**，靠传播与文档 |
| **我们已有** | `marketplace.json`、`plugin/hooks/hooks.json` |
| **动作** | 用户指南增加 Claude Code 插件一节；**不要把插件当 Skill 主入口**（避免吓跑只想装 Markdown 的用户） |

| **动作** | `npx skills add … -a cursor` 或复制到 `~/.cursor/skills/` |

```markdown
## What
RuleDoctor is an Agent Skill that makes coding agents read `CLAUDE.md`, `.cursorrules`, and configurable `required_reads` before editing—plus optional shell guards for `git push --force`.

- Repo: https://github.com/syf2211/ruledoctor
- Skill path: `skills/ruledoctor` (subdirectory `skills`)
- skills.sh: https://skills.sh/syf2211/ruledoctor/ruledoctor

Deep link (Claude): `ccswitch://v1/import?resource=skill&app=claude&name=ruledoctor&repo=syf2211/ruledoctor&directory=skills&branch=main`

## Why
Many users already maintain project rules but agents skip them after compaction. This skill is lightweight (SKILL.md only) and complements CC Switch’s workflow.

## Ask
Consider adding this repo to the built-in skill repository list so users can discover it without manual repo setup.

Thanks!
```
